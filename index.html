Hello World, Let There Be Hexo!

Welcome to my Hexo blog! I’ve found Hexo to be an efficient, robust blog engine you can run with NodeJS and npm. I recommend this engine to anyone looking to blog with the Node community’s support. Check the GitHub documentation for Hexo on the published GitHub Pages site (Hexo hooks in nicely to GitHub and GitHub Pages). If you’re acquainted with Node and npm at all, startup is painless and I’ll walk you through it here. However, if you run into any problems while setting up or using Hexo, you can find the Hexo crew’s answers in the troubleshooting documentation or you can ask me on Twitter.

Here we go: Quick Start!

Requirements

1. Git
2. Chrome’s V8 JavaScript engine.
3. Node.js - (https://nodejs.org/)
4. npm - (https://docs.npmjs.com/getting-started/installing-node)
