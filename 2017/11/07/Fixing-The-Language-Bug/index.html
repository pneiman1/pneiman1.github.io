Fixing The Language Bug (Quickly!)


Perhaps you have noticed by now, if you have run the Hexo server multiple times, the categories, tags, titles, etc., of your site fluctuate between languages, e.g., one second they’re French, the next they’re German. This is an internationalization bug, wherein a failure to revert to the default occurs repeatedly. This is a known bug in the Landscape theme as well as the Light theme.
